# Comandos Git - Sara Bastidas

## 1. Crear una nueva rama local con mi nombre
```bash
git checkout -b saraBastidas
```

## 2. Crear un nuevo commit en esa rama local
```bash
git add .
git commit -m "Agrega avances de Sara Bastidas"
```

## 3. Subir la rama local al repositorio remoto
```bash
git push -u origin saraBastidas
```
